# Wispr MR Website

Static landing page ready for Vercel.

## Deploy on Vercel

1. Push this repo to GitHub.
2. In Vercel, import the repo.
3. Set **Root Directory** to `website`.
4. Keep build settings empty/default:
   - Build Command: empty
   - Output Directory: empty
5. Deploy.

The Windows download button points to:

```text
/downloads/WisprMR-Setup.exe
```

The macOS download button points to:

```text
/downloads/WisprMR-Install.zip
```

If the app package changes, copy the new artifacts to:

```text
website/downloads/WisprMR-Setup.exe
website/downloads/WisprMR-Install.zip
```
